### Redis基本介绍
- 安装
  ```bash
  #Myenv:Amazon Linux AMI release 2017.09
  rpm -ivh http://dl.fedoraproject.org/pub/epel/6/x86_64/epel-release-6-8.noarch.rpm
  yum --enablerepo=epel -y install redis
  ```
- Redis数据结构
  共支持5种数据结构，包括：
  >STRING（字符串）
  >LIST（列表）
  >SET（集合）
  >HASH（散列）
  >ZSET（有序集合）
